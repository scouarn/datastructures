# Data structure library

Collection of various handy cross-platform pieces of code about data manipulation implemented in C (ansi standard) with an homogeneous interface.

## What's done :

* error utility
* hashtable
* stack
* matrix
* dict
* bitfield
* queue
* threadpool
* arbitrary precision int
